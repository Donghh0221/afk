# Next.js Project

This is a Next.js 15 project managed by AFK.

## Tech Stack

- Next.js 15 (App Router)
- React 19
- TypeScript
- Tailwind CSS

## Guidelines

- Use the App Router (`app/` directory)
- Server Components by default; add `"use client"` only when needed
- Use Tailwind CSS for styling
- Run `npm run build` to verify before completing

## Getting Started

If this is a fresh project, initialize with:
```bash
npx create-next-app@latest . --typescript --tailwind --app --src-dir --no-import-alias
```
